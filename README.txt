Hosam Reda personal website

Open index.html in your browser to view the site.

What is included:
- Personal hero section for Hosam Reda.
- Light and dark mode toggle.
- 12-hour live clock.
- Arabic date.
- Prayer times for Cairo, Egypt using Aladhan API when internet is available.
- Fallback prayer times if the API cannot be reached.
- Morning and evening azkar tabs.
- Services and contact sections.
- Facebook search/contact link for "Hosam Reda Asc".

Main files:
- index.html: page content
- styles.css: visual design and responsive layout
- script.js: clock, prayer times, azkar, theme toggle, and animated background
- assets/hosam-profile.jpeg: main profile image
- assets/hosam-secondary.jpeg: secondary image
